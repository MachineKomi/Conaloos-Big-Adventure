# Character remixes

A remix is a sprite combination that visually creates a "new" character: a butterfly wearing a house, a peep with a butterfly hat, the moon turning into a face. Each remix is documented here so the next agent doesn't reinvent (or accidentally clash with) one.

Format:

```
## {short title}
- **Visual recipe:** which sprites, layered how (z-order, scale, anchor).
- **Where it appears:** scene + hotspot id.
- **What it does:** the rare-response rhyme and any persistence behaviour.
```
